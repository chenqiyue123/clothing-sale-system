# 服装管理系统 README

## 项目概述
本服装管理系统是一个基于C语言开发的控制台应用程序，旨在提供便捷的服装商品信息管理功能，包括商品录入、查询、修改、删除、购物车操作及销售数据分析等，适用于小型服装店铺的日常运营管理。

## 核心功能
程序通过主菜单提供以下功能选项：

1. **录入服装**：按指定格式批量输入服装信息（品牌、类别、售价、销量、成本、库存），空行结束录入
2. **浏览服装**：展示所有有效服装的完整信息，包括自动计算的利润率
3. **删除服装**：支持单个/批量删除指定编号服装，输入"全部删除"可清空数据
4. **修改服装**：通过"字段:值"格式修改指定编号服装的信息（如"售价:399"）
5. **加入购物车**：批量添加指定编号服装到购物车（格式示例："0 2 3"）
6. **查看购物车**：显示购物车中所有商品及数量、小计和总金额
7. **结算**：确认购物车商品并完成结算，自动更新库存和销量
8. **直接结算**：不经过购物车，直接选择商品完成结算流程
9. **价格区间利润率**：统计<100、100-300、>300三个价格区间的利润率
10. **因素影响分析**：分析品牌、类别、价格区间对销量的影响（计算平均销量）
11. **品牌业绩分析**：统计各品牌总利润、主打类别及其利润占比
0. **退出系统**：保存数据并退出程序

## 数据结构
- 服装信息存储在`Cloth`结构体数组中，最大支持200条记录
- 每条服装记录包含：品牌、类别、售价、销量、成本、库存及有效性标识
- 购物车采用编号数组实现，支持记录多个商品的选中状态

## 数据存储
所有服装数据以文本格式保存在`cloth.txt`文件中，每条记录格式为：
```
品牌 类别 售价 销量 成本 库存
```
示例：
```
Nike Jacket 599.0 25 299.0 40
Uniqlo Shirt 149.0 89 79.0 120
```

## 操作说明
1. 运行程序后，根据菜单提示输入功能编号进行操作
2. 批量操作时（如批量删除、添加购物车），使用空格分隔多个编号
3. 修改操作需严格遵循"字段:值"格式，支持的字段包括：品牌、类别、售价、销量、成本、库存
4. 结算前系统会自动检查库存是否充足，库存不足时将取消操作
5. 所有数据修改会实时保存到文件中，确保数据不丢失

## 编译与运行
使用C编译器编译源代码：
```bash
gcc 25215120421陈启粤项目代码.cpp -o cloth_system
```
运行生成的可执行文件：
```bash
# Linux/Mac系统
./cloth_system

# Windows系统
cloth_system.exe
```

## 注意事项
1. 服装编号由系统自动分配（0-199范围），不可手动修改
2. 库存为0的服装会被标记为无效，不参与正常展示和销售
3. 输入数据时请遵守格式要求，否则可能导致操作失败
4. 程序支持中文显示，确保运行环境的编码设置正确

## 系统特点
- 轻量级设计，无需数据库支持，通过文本文件存储数据
- 操作便捷，支持批量处理，提高管理效率
- 内置数据分析功能，帮助用户了解销售情况和利润表现
- 自动数据校验，防止无效数据录入和不合理操作
