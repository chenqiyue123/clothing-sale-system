# Clothing Sale System · 服装销售管理系统

C 语言控制台程序，提供服装商品管理、销售分析和购物车功能。

## ✨ 功能

- 📦 **商品管理**：录入、查询、修改、删除服装信息
- 🛒 **购物车**：添加商品、计算总价
- 📊 **销售分析**：销量统计与数据分析

## 🛠 技术栈

C · 控制台程序 · 文件存储

## 🚀 快速开始

```bash
git clone https://github.com/chenqiyue123/clothing-sale-system.git
cd clothing-sale-system
gcc ClothingSale.cpp -o ClothingSale
./ClothingSale
```
